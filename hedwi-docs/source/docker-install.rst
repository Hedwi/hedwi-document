使用 docker 部署（推荐）
------------------------

docker 安装
=====================

.. code-block:: bash

    curl 'https://hedwi.com/install' |sh

启动
============

.. code-block:: bash

    cd hedwi_pkgs/compose/docs && docker compose up -d

访问
============

    访问 http://127.0.0.1:40008 即可
